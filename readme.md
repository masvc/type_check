# 2×2タイプ診断サービス

## 1. 概要
本診断サービスは「陽キャ・陰キャ」×「根クラ・根アカ」の2×2タイプ分類を行うWebアプリケーションです。
ユーザーは15問の質問に回答し、選択肢ごとに加算されるポイントによって最終的なタイプを診断します。
結果はChart.jsを使用して視覚化され、キャリア相談リンクおよびSNSシェアリンクが設置されます。

## 2. 技術スタック
- React + Vite
- TypeScript
- TailwindCSS
- Chart.js（診断結果の可視化）
